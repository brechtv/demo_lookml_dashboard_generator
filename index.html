1, 2, 3 Dashboard

Generates dashboard LookML for a Business Pulse dashboard built on thelook, with a custom header and color palette.

How to

Add in your color palette and generate the dashboard YAML.

Copy the output and create a new dashboard in a project that's built on thelook. Make sure you have an include: "*.dashboard" on the model file.

Save your dashboard (no need to commit), and hit 'view dashboard' from the LookML file. Now you can view the dashboard and hit 'copy to space' to convert it into a user-defined dashboard!
